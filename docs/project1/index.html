AI 시대, 비개발자를 위한 개발 지식

AI가 코드를 짜주는 시대, 왜 우리는 여전히 개발을 알아야 할까요?
AI와 협업하여 '내 아이디어'를 현실로 만들기 위한 최소한의 지식을 알아봅니다.

키보드 방향키 또는 화면의 버튼으로 시작하세요.

GUI vs CLI

우리가 매일 사용하는 '클릭'과 '드래그'의 세상(GUI)과, 개발자들이 '명령어'를 입력하는 세상(CLI)의 차이를 알아봅니다.

GUI (Graphical User Interface)

아이콘, 버튼, 창 등 그래픽을 통해 컴퓨터와 소통하는 방식. 직관적이고 사용하기 쉽습니다.

CLI (Command Line Interface)

텍스트 명령어를 입력하여 컴퓨터를 직접 제어하는 방식. 개발 환경 설정, 패키지 설치 등 강력한 작업에 필수적입니다.

터미널: 컴퓨터와 대화하는 창구

CLI 명령어를 입력하는 곳이 바로 '터미널'입니다. AI가 "이 명령어를 입력하세요"라고 할 때, 바로 이곳을 사용합니다.

# AI가 이런 명령어를 알려줄 때...
pip install pandas

# ...터미널에 입력하면 필요한 도구가 설치됩니다.
# Windows: cmd 또는 PowerShell
# macOS: Terminal 앱

VS Code 같은 코드 편집기 내에도 터미널 기능이 내장되어 있습니다.

언어 vs 프레임워크

프로그래밍 '언어'는 재료, '프레임워크'는 요리법(레시피)과 같습니다.
효율적인 개발을 위해 미리 짜놓은 구조가 프레임워크입니다.

프로그래밍 언어

컴퓨터와 소통하기 위한 문법 규칙.

JavaScript Python Java

프레임워크

언어를 사용해 효율적으로 개발하기 위한 뼈대/구조.

React (JS) Next.js (JS) Django (Python)

소프트웨어 작동 원리 (간단히)

우리가 보는 화면(프론트엔드)과 보이지 않는 곳에서 일하는 서버(백엔드), 그리고 정보를 저장하는 창고(DB)로 구성됩니다.

프론트엔드 (클라이언트)

백엔드 (서버)

데이터베이스 (DB)

로컬 개발에서 배포까지

내 컴퓨터(로컬)에서 만든 프로그램을 다른 사람들도 쓸 수 있도록 인터넷 세상(서버)에 공개하는 과정을 '배포'라고 합니다.

로컬(Local) 환경

개발자의 개인 컴퓨터. 여기서 코드를 작성하고 테스트합니다.

배포(Deployment)

작성된 코드를 AWS, Vercel 같은 클라우드 서버에 올려 인터넷을 통해 누구나 접속할 수 있게 만듭니다.

개발 환경 세팅하기

AI와 함께 개발을 시작하기 위해 필요한 기본 도구들입니다.

IDE (코드 편집기)

Cursor, VS Code 등 코드를 작성하고 AI와 대화하는 공간.

패키지 매니저

`pip`(Python), `npm`(Node.js) 등 유용한 코드 묶음(라이브러리)을 쉽게 설치.

Github

코드의 버전을 관리하고 다른 사람과 협업하기 위한 필수 서비스.

컴퓨터가 좋아하는 데이터 형식

AI에게 정보를 효과적으로 전달하려면 컴퓨터가 이해하기 쉬운 형식(Markdown, JSON, CSV)을 사용하는 것이 좋습니다.

JSON (JavaScript Object Notation)

{
  "이름": "지민",
  "나이": 14,
  "학교": "강남중"
}

Markdown

# 제목 (h1)
- 리스트 항목 1
- **굵은** 글씨
- *기울인* 글씨

AI와 협업('바이브 코딩') 잘하는 법

AI에게 일을 맡길 때, 더 좋은 결과를 얻기 위한 몇 가지 실용적인 팁입니다.

1. 계획부터 세우고 허락받기

"바로 만들지 말고, 어떻게 만들지 계획부터 설명해줘."
AI가 마음대로 파일을 망가뜨리는 것을 방지할 수 있습니다.

2. 명확한 규칙(Rules) 설정하기

"언어는 Python 3.10 버전을 사용해." 와 같이 반복적인 지침을 문서로 저장해두면 AI가 일관성 있게 작업합니다.

3. Github로 자주 저장하기 (Commit & Push)

작업 단계를 자주 저장하여 AI의 실수나 예기치 않은 오류로부터 안전하게 코드를 보호하고, 언제든 이전 버전으로 돌아갈 수 있습니다.

이제, 당신의 아이디어를 만들 시간

개발 지식은 더 이상 개발자만의 전유물이 아닙니다.
기초를 이해하고 AI라는 강력한 도구를 활용하여, 상상했던 것을 직접 만들어보세요.

이 프레젠테이션은 요청하신 아티클과 디자인 가이드라인을 기반으로
HTML, CSS, JavaScript를 사용하여 제작되었습니다.
